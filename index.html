🎯 게임 가이드

1. 몸으로 말해요 (10:50~11:08)
팀에서 한명이 나와서 제시어를 몸으로 표현하여 3명의 팀원이 맞추는 게임
• 10:50~10:53: 게임 설명 (3분)
• 10:53~11:08: 게임 진행 (15분, 1팀당 2분 제한)
• 11:08~11:10: 휴식 및 다음 게임 준비 (2분)
2. 발목 풍선 터뜨리기 (11:10~11:27)
고무줄을 이용해서 발목에 풍선을 달아서 서로 팀 대항전으로 상대팀의 풍선을 터뜨리는 게임
• 11:10~11:15: 게임 설명 (5분)
• 11:15~11:27: 게임 진행 (12분, 2팀씩 대결 토너먼트, 1경기당 3분)
• 11:27~11:30: 휴식 및 다음 게임 준비 (3분)
• 승리조건: 모든 풍선을 터뜨리면 승리
3. 단합 퀴즈 (11:30~11:50)
어려운 퀴즈를 준비하고 1번부터 4번까지 보기를 준비한 다음 팀원끼리 상담해서 최종 답안을 정하는 게임
• 11:30~11:32: 게임 설명 (2분)
• 11:32~11:50: 게임 진행 (18분, 1팀당 3분의 시간동안 상의 가능)
• 보다 많이 맞힌 팀이 승리
•
4. 카드 뒤집기 (13:00~13:30)
쌍이 존재하는 카드를 뒤집어두고 한 팀에 한명씩 나와서 카드 2개를 뒤집어서 쌍이 맞는 경우 획득
• 13:00~13:10: 카드 세팅 (10분)
• 13:10~13:15: 게임 설명 (5분)
• 13:15~13:30: 게임 진행 (15분, 2팀씩 대결, 3분 제한, 카드 고르는 시간 10초)
• 카드 예시: "가는말이 고아야 - 오는말이 곱다", "ㅂㅂㅂ - 비빔밥", "티끌 모아 - 태산"
5. 등을 맞대고 그림 그리기 (13:30~13:45)
각 팀별로 한 줄로 의자에 앉아서 첫번째 사람한테 간단한 그림을 보여주고, 앞사람 등에 그림을 그려서 전달
• 13:30~13:33: 게임 설명 (3분)
• 13:33~13:45: 게임 진행 (12분, 1명당 20초, 4명 기준 1분 20초 + 의자 배치 40초)
• 13:45~13:50: 마지막 게임 준비 (5분)
• 그림 예시: 집, 강아지, 꽃 등 간단한 그림
6. 오타마 릴레이 (13:50~14:10)
모든 팀이 동시에 숟가락에 탁구공을 올리고 반환지점을 돌아서 다시 돌아와서 다음 사람에게 넘기는 게임
• 13:50~13:55: 게임 설명 (5분)
• 13:55~14:10: 게임 진행 (15분, 2팀씩 대결 후 3팀 결승, 시간제한 2분)
• 주의사항: 탁구공을 떨어뜨리면 다시 시작
• 모든 팀원이 다했을 때 승리

🏟️ 토너먼트

토너먼트 방식
6팀 기준:
• 1라운드: 2팀씩 3경기 → 승자 3팀 진출
• 결승전: 3팀이 동시에 경기하여 1,2,3등 결정

점수: 1등 5점, 2등 3점, 3등 1점이 자동으로 추가됩니다.

팀을 먼저 설정한 후 토너먼트를 시작하세요.

💬 제시어 & 그림 관리

사용법
제시어 게임: 6경기로 구성되며, 각 경기당 20개의 제시어가 제공됩니다.
게임 진행: 2분 제한시간 동안 제시어를 맞추고 정답/스킵 버튼을 활용하세요.
제시어 섞기: 매 게임 시작 시 해당 경기의 제시어 순서가 랜덤하게 섞입니다.
전체 셔플: 아래 셔플 버튼으로 모든 제시어를 섞어서 6경기에 새롭게 분배할 수 있습니다.
일본어 번역: 각 제시어마다 일본어 번역이 함께 제공됩니다.
단축키: ESC 키로 게임을 종료할 수 있습니다.

제시어 전체 셔플

현재 상태: 기본 카테고리별 구성

• 기본: 1경기(동물), 2경기(교통수단), 3경기(스포츠용품), 4경기(전자기기), 5경기(음식), 6경기(생활용품)
• 셔플: 모든 제시어를 섞어서 각 경기에 랜덤하게 20개씩 새로 배치
